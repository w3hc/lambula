# Lambula

## Install

```bash
pnpm i
```

## Run

Create a `.env` file:

```
cp .env.example .env
```

Add your own keys in the `.env` file (you can get it in your [Reown dashboard](https://cloud.reown.com/)), then:

```bash
pnpm dev
```
